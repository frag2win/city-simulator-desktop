# Security Policy

## Supported Versions

Only the latest version of City Simulator is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v2.x    | :white_check_mark: |
| < v2.0  | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities to shubhmapawar@gmail.com. We will acknowledge your report within 48 hours and keep you informed of the progress.
